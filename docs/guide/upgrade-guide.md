# Upgrade Guide

## Semantic Versioning
VitePress DepthIndex follows Semantic Versioning rules:
- **Major changes (x.0.0)**: Breaking API adjustments.
- **Minor changes (1.x.0)**: New features and enhancements (e.g. adding PWA, localization, or delta indexing).
- **Patch updates (1.1.x)**: Performance optimizations and bug fixes.

## Before Upgrading
Before upgrading DepthIndex:
- Backup your configuration file `.vitepress/config.ts`.
- Clear browser caches to remove old service worker registrations.
- Re-run local test environments: `npm run dev`.

## v1.0.x → v1.1.x
Upgrading to v1.1.x introduces:
- **PWA Service Worker** (`depthindex-sw.js`): Generated automatically in the output folder.
- **Plugin SDK**: Exposes hooks for customizing search and synthesis.
- **Bilingual translation**: English (`en`) and Tagalog (`tl`) support.

## v1.1.x → v1.2.0
*(Note: As of version `1.1.6`, version `1.2.0` features like secure update signatures and local analytics are included as stable updates in the `1.1.6` codebase).*
Upgrading to version `1.1.6` introduces:
- **Delta Indexing**: Differential indexing that reduces index update payload sizes.
- **Privacy Firewall**: Automated PII scrubbing and restricted analytics metrics.
- **Device Adaptation**: Profiling system that automatically scales search settings.

## Breaking Changes
- **Personalization Defaults**: Starting in `v1.1.x`, personalization is **disabled by default**. To keep history tracking enabled, set `personalization.enabled: true` in your configuration.
- **Service Worker Registrations**: Service workers are now registered relative to the base URL path, which may require path adjustments on subfolder deployments.

## New Features by Version
- **v1.1.6**: Privacy Firewall, Delta Indexing, Secure updates, Device Adaptation, Tagalog translations.
- **v1.1.5**: Suggestion Engine, Search Mode manager, NLU intent classification.
- **v1.1.0**: Plugin SDK, compliance checking, bilingual translation engine.
- **v1.0.0**: Initial release (BM25 keyword search, on-device vector calculations).

## Deprecated Features
- **Legacy UI styles**: Styling properties in the `ui` object (e.g. `ui.position`, `ui.triggerIcon`) are deprecated. Use the `panel` and `floatingButton` configuration blocks instead.
- **SiteContextEngine Questions**: Welcome screen suggestions are now generated by `SuggestionEngine`.

## Migration Checklist
Follow these checks when upgrading to `1.1.6`:
- [ ] Update dependency version in `package.json` to `"vitepress-plugin-depthindex": "^1.1.6"`.
- [ ] Run a clean install: `npm install`.
- [ ] Update configurations: replace deprecated `ui` options with `panel` and `floatingButton` options.
- [ ] Verify environment variables: ensure `VITE_DEPTHINDEX_CLOUD_API_KEY` is set correctly.
- [ ] Clear browser storage: delete the `depthindex_secure_store` IndexedDB instance to avoid schema mismatches.

## Rollback
To roll back to a previous version:
1. Revert the dependency version in `package.json`.
2. Clear the service worker cache in your browser.
3. Restart your dev server: `npm run dev`.
